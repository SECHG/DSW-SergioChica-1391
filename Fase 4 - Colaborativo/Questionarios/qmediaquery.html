Responda las siguientes preguntas

1. ¿Qué propiedad de CSS se utiliza para definir las reglas de estilo específicas para diferentes tamaños de pantalla?
A. display
B. width
C. font-size
D. media
2. ¿Cuál es la sintaxis correcta para definir una media query en CSS?
A. @media (max-width: 768px) { ... }
B. @media { max-width: 768px; ... }
C. media (max-width: 768px) { ... }
D. (max-width: 768px) @media { ... }
3. ¿Qué significa el término "breakpoint" en el contexto de las media queries?
A. El punto en el que la página web se rompe y deja de funcionar correctamente
B. El punto en el que se cambia la orientación de la pantalla del dispositivo
C. El punto en el que se aplica una regla de estilo específica para un tamaño de pantalla determinado
D. El punto en el que se carga el contenido multimedia de la página web
4. ¿Qué tipo de unidades se recomiendan para definir las media queries?
A. Em y porcentaje
B. Píxeles y centímetros
C. Puntos y pulgadas
D. Milisegundos y segundos
5. ¿Cuál es la forma más común de utilizar las media queries en la actualidad?
A. Definir reglas de estilo específicas para cada dispositivo y tamaño de pantalla
B. Utilizar un diseño fluido que se adapte a cualquier tamaño de pantalla
C. Utilizar un diseño responsivo que se adapte a diferentes tamaños de pantalla mediante media queries
D. Utilizar un diseño estático que no cambia en función del tamaño de pantalla
